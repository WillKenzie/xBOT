# xBot
A simple discord bot.
## Commands.  
* !exit  
exits the bot.  
* !time  
tells the time.  
* !numgen  
Generate a random number between 0 and 1000.  
* !coinflip  
flips a coin.  
* !say  
Say something as the bot!  
* !sophie (report)
Shows an embed about Sophie and a link to the report form.
